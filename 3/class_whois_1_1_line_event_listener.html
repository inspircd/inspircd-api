Called whenever a line of WHOIS output is sent to a user. You may change the numeric and the text of the output by changing the values numeric and text, but you cannot change the user the numeric is sent to.

Parameters
whoisWhois context, can be used to send numerics
numericNumeric being sent
Returns
MOD_RES_DENY to drop the line completely so that the user does not receive it, or MOD_RES_PASSTHRU to allow the line to be sent.
