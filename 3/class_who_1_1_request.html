virtual bool Who::Request::GetFieldIndex ( char  flag,
size_t &  out 
) const
pure virtual

Get the index in the response parameters for the different data fields

The fields 'r' (realname) and 'd' (hops) will always be missing in a non-WHOX query, because WHOX splits them to 2 fields, where old WHO has them as one.

Parameters
flagThe field name to look for
outThe index will be stored in this value
Returns
True if the field is available, false otherwise
